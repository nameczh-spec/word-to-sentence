# 不背单词风格功能 - 产品需求规范

## 1. 概述

### 1.1 项目背景
为"单词化句"项目添加类似"不背单词"App的完整学习功能，提供科学记忆、多种练习模式、深度词典查询。

### 1.2 核心功能范围

#### 必须实现
- ✅ 记忆曲线复习（多种内置 + 自定义）
- ✅ 默写练习模式（键盘输入 + 手写输入）
- ✅ 拼写练习
- ✅ 听写练习（音频）
- ✅ 组词练习
- ✅ 词根词缀系统
- ✅ 近反义词查询
- ✅ 派生词关联

#### VIP功能（用户要求实现）
- ✅ 语境选择题
- ✅ 单词配对
- ✅ 派生连句
- ✅ 柯林斯词典（简化版）

#### 数据来源
- 本地数据：词根词缀库、派生词映射、已有真题数据
- API：有道翻译API（用户提供的代理）
- 搜索获取：补充词根词缀数据

## 2. 功能详情

### 2.1 默写练习模式（Pad端专属）

#### 输入方式
1. **键盘输入**
   - 传统键盘/虚拟键盘
   - 自动判定正确与否
   - 实时显示 ✓/✗

2. **手写输入**
   - 支持平板触控笔/手指
   - 无自动判定
   - 用户自行对照答案
   - 提供橡皮擦、撤销、清空工具

#### 默写模式
- **默写单词**：左侧显示释义，右侧填写单词
- **默写释义**：左侧显示单词，右侧填写释义
- **默写音标**：左侧显示单词，右侧填写音标

#### 交互流程
```
键盘模式：
1. 点击输入框 → 输入
2. 回车确认 → 自动判定显示 ✓/✗
3. 自动进入下一个
4. 可随时点击"显示"/"跳过"
5. 点击"交卷"结束

手写模式：
1. 点击手写区域 → 书写
2. 点击"对照答案" → 显示正确答案
3. 用户自行判断 → 点击"我写对了"/"我写错了"
4. 可点击"跳过"
5. 点击"交卷"查看完整对比
```

#### 与记忆曲线联动
```
错误次数影响复习安排：
├── 错误1次 → 间隔×1（正常）
├── 错误2次 → 间隔×0.5（缩短）
└── 错误3次+ → 标记困难词，每日必复习

默写结果同步：
├── 正确 → 阶段+1
├── 错误 → 阶段回退
└── 结果保存到 learningProgress
```

### 2.2 其他练习模式

| 模式 | 说明 | 输入方式 |
|------|------|----------|
| 拼写练习 | 英文拼写测试 | 键盘 |
| 听写练习 | 音频播放后拼写 | 键盘 |
| 组词练习 | 拖拽字母组成单词 | 触摸/拖拽 |
| 语境选择 | 根据例句选正确单词 | 点击选择 |
| 单词配对 | 匹配单词和释义 | 拖拽配对 |
| 派生连句 | 根据语境填空 | 选择/输入 |

### 2.3 词典功能

#### 词根词缀
- 本地词根库（2000+词根词缀）
- 展示词根、词缀、派生词
- 溯源解义

#### 派生词
- 本地派生词映射
- API补充获取
- 同根词关联

#### 近反义词
- 有道API获取
- 用户代理支持

#### 柯林斯词典
- 有道API / 本地简化版
- 英英释义
