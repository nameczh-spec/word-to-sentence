# 考研英语真题翻译内容完善需求文档

> 目标：为所有43套真题补充完整的翻译参考译文和句子解析
> 当前状态：翻译数据质量参差不齐，格式不统一，多数缺少参考译文和解析

---

## 一、当前问题诊断

### 1. 数据格式不统一

| 字段名 | 存在情况 | 说明 |
|--------|----------|------|
| `text` | 大部分年份 | 只有英文原文 |
| `original + translation` | 2000/2001/2005等少数年份 | 有原文和译文 |
| `rawText` | 大部分年份 | 整段原文（包含广告水印） |

### 2. 内容质量问题

- **缺少参考译文**：约40套试卷只有英文原文，没有中文译文
- **句子划分错误**：有些把整段话都包含在一个句子里，不只是划线句子
- **数据来源混乱**：有的带有淘宝店铺广告水印
- **缺少句子解析**：没有句型分析、重点词汇、翻译技巧等内容

### 3. 翻译模块期望的数据格式

翻译模块 `translation.js` 中的 `translationDB` 对象格式：

```javascript
translationDB = {
    "英文原句全文": {
        translation: "中文参考译文",
        analysis: `HTML格式的句子解析内容`
    }
}
```

## 四、内容要求

### 参考译文要求

1. **准确性**：忠实于原文，意思准确无误
2. **通顺性**：符合中文表达习惯，语句流畅
3. **翻译腔**：避免生硬的直译，适当调整语序
4. **要点齐全**：每个信息点都要翻译到位

### 句子解析要求

每个句子的解析应包含三部分：

1. **句型结构**（2-3个要点）
   - 分析句子的主要语法结构
   - 指出从句类型、特殊句型等
   - 说明句子的逻辑关系

2. **重点词汇**（3-5个单词/词组）
   - 列出句中的重点或难点词汇
   - 给出准确的中文释义
   - 标注词性

3. **翻译技巧**（2-3个要点）
   - 说明翻译时采用的技巧
   - 指出容易译错的地方
   - 提供更好的译法建议

## 五、输出格式建议

### 格式一：直接写入JSON（适合程序处理）

```json
{
  "num": 46,
  "original": "Scientists jumped to the rescue with some distinctly shaky evidence to the effect that insects would eat us up if birds failed to control them.",
  "translation": "科学家们急忙出来救驾，提出了明显站不住脚的证据，大意是说如果鸟类不能控制昆虫的数量，昆虫便会把我们吞噬殆尽。",
  "analysis": {
    "structure": [
      "主句结构：Scientists jumped to the rescue... 主谓结构，jumped to the rescue 是固定搭配",
      "with 介词短语作方式状语：with some distinctly shaky evidence 说明"救驾"的方式",
      "that 引导同位语从句：that insects would eat us up... 解释说明 evidence 的具体内容"
    ],
    "vocabulary": [
      {"word": "jump to the rescue", "meaning": "赶来救援，急忙救驾"},
      {"word": "distinctly", "meaning": "adv. 明显地，确实地"},
      {"word": "shaky", "meaning": "adj. 不可靠的，站不住脚的"},
      {"word": "to the effect that", "meaning": "大意是说"},
      {"word": "eat up", "meaning": "吃光，吞噬"}
    ],
    "techniques": [
      "jump to the rescue 不宜直译为"跳到救援"，译为"急忙出来救驾"更形象生动",
      "shaky evidence 译为"站不住脚的证据"，符合中文表达习惯",
      "同位语从句单独成句，用"大意是说"引导，更符合中文表达"
    ]
  }
}
```

### 格式二：Markdown文档（适合人工生成后程序提取）

```markdown
## 2025年英语一翻译

### 第46题
**原文**：Scientists jumped to the rescue with some distinctly shaky evidence...

**译文**：科学家们急忙出来救驾，提出了明显站不住脚的证据...

**句型结构**：
1. 主句结构：Scientists jumped to the rescue...
2. with 介词短语作方式状语
3. that 引导同位语从句

**重点词汇**：
- jump to the rescue：赶来救援，急忙救驾
- distinctly：adv. 明显地，确实地
- shaky：adj. 不可靠的，站不住脚的

**翻译技巧**：
1. jump to the rescue 译为"急忙出来救驾"更形象
2. shaky evidence 译为"站不住脚的证据"
```

## 六、注意事项

1. **句子划分要准确**：只翻译划线的句子，不要包含整段内容
2. **译文质量要高**：不能是机翻水平，要有考研英语辅导的质量
3. **解析要实用**：针对考研翻译考点，不要泛泛而谈
4. **格式要统一**：所有年份的格式要保持一致
5. **校对原文**：先确认英文原文是否准确（有些带广告水印需要清理）
6. **年份顺序**：建议从近年往早年做，近年考生更关注

## 七、与作文任务的优先级对比

| 任务 | 工作量 | 重要性 | 建议优先级 |
|------|--------|--------|------------|
| 作文范文补充 | 大（82篇作文） | 高 | 先做近5年 |
| 翻译译文补充 | 中（约200个句子） | 高 | 与作文交替进行 |
| 翻译句子解析 | 很大（每句3个模块） | 中 | 后做，或只做近10年 |
