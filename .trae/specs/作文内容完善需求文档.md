## 一、数据结构说明

作文范文数据存储在 `vocab-reading-generator/js/writing.js` 的 `sampleData` 对象中。

### 整体结构

```javascript
sampleData = {
    '年份-类型编号': {
        partA: { /* 应用文部分 */ },
        partB: { /* 大作文部分 */ }
    }
}
```

**key规则**：
- 英语一：`{year}-1`（如 `2023-1`）
- 英语二：`{year}-2`（如 `2023-2`）
- 2010年以前（不分英一英二）：`{year}-1`（如 `2005-1`）

### Part A（应用文）结构

```javascript
partA: {
    title: '作文题目中文标题',
    translation: '题目中文翻译/说明',
    samples: [
        {
            text: '范文正文内容',
            note: '范文点评说明'
        }
    ],
    tips: {
        type: 'email / letter / notice / ...',
        structure: ['结构要点1', '结构要点2', '结构要点3'],
        commonPhrases: ['常用表达1', '常用表达2'],
        notes: '写作注意事项说明'
    }
}
```

### Part B（大作文）结构

```javascript
partB: {
    title: '作文题目中文标题',
    translation: '题目中文翻译/说明',
    type: 'picture / chart / table / ...',
    samples: [
        {
            text: '范文正文内容',
            note: '范文点评说明'
        }
    ],
    tips: {
        type: 'picture / chart / table / ...',
        structure: ['结构要点1', '结构要点2', '结构要点3'],
        keyExpressions: ['关键表达1', '关键表达2'],
        notes: '写作注意事项说明'
    }
}
```

## 二、内容要求

### 范文要求

1. **字数**：
   - Part A：约100词
   - Part B：约160-200词

2. **质量**：
   - 语言地道，符合考研英语写作规范
   - 结构清晰，逻辑连贯
   - 句式多样，词汇丰富
   - 紧扣题目要求，要点齐全

3. **数量**：每篇作文至少1篇范文，建议2篇（不同风格/角度）

### 写作技巧要求

1. **type字段**：作文类型（email, letter, notice, picture, chart, table等）
2. **structure**：3-5个结构要点，描述文章的段落结构
3. **commonPhrases / keyExpressions**：5-10个常用表达或关键句型
4. **notes**：1-2句话的写作注意事项

## 七、注意事项

1. 范文内容必须紧扣题目，不得偏题
2. 语言要符合考研英语写作水平，不能过于简单也不能过于生僻
3. 结构要有逻辑，段落分明
4. 写作技巧要实用，对考生有实际帮助
5. 注意格式规范，缩进和换行要正确
6. 范文中的换行用反引号字符串（`...`）保留格式
7. 2000-2009年的试卷，key格式为 `'2005-1'`（对应英语一），即使只有Part B也要按此格式
