### 3. `parseContent()` - `（中文释义）` 清理（BUG-4）

**位置**：`apiRequest.js` 第 194 行

```javascript
// 旧：字符类过窄，无法匹配词性、分号、英文等
text = text.replace(/\s*（[\u4e00-\u9fff\w、，。]+）/g, '');

// 新：使用排除法，匹配右括号前的所有内容
text = text.replace(/\s*（[^）]+）/g, '');
```

**修复效果**：
- ✅ `（n. 数据库）` → 清理成功
- ✅ `（数据库；数据存储）` → 清理成功
