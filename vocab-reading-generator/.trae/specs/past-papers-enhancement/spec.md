# 考研真题模块功能增强 - 产品需求文档

## Overview
- **Summary**: 对现有考研真题练习模块进行全面功能增强和界面优化，包括布局调整、翻译/作文模块完善、长难句数据扩充、做题模式工具栏、真题精读交互修复等
- **Purpose**: 提升真题练习模块的完整性和用户体验，使功能更加完善实用
- **Target Users**: 考研英语学习者

## Goals
- 优化历年真题列表布局，采用网格布局提升空间利用率
- 完善翻译练习模块，实现数据提取和完整交互
- 完善作文指导模块，实现数据提取和完整交互
- 大幅扩充长难句数据量（至少100句以上，越多越好）并规范解析格式
- 新增做题模式工具栏（选择模式、书写模式、橡皮擦）
- 修复真题精读题目交互，支持选项选择

## Non-Goals (Out of Scope)
- 不新增作文在线评分功能
- 不新增翻译在线评分功能
- 不做用户数据统计分析
- 不做移动端适配优化（已有移动端版本）

## Background & Context
现有真题练习模块已具备真题精读、完形填空、长难句分析三个功能，但存在以下问题：
1. 历年真题列表每年一个分组，空间利用率低
2. 翻译和作文模块只有入口卡片，点击后无实际内容
3. 长难句数据量太少（仅11句），无法满足学习需求
4. 缺少做题模式切换工具栏
5. 真题精读中的题目无法点击选择答案

## Functional Requirements

### FR-1: 历年真题网格布局
- 将历年真题列表从按年分组的纵向布局改为网格布局
- 所有试卷卡片并排显示，一行多个，自动换行
- 保持试卷卡片的悬停效果和点击交互

### FR-2: 翻译练习模块完善
- 从真题JSON数据中提取翻译部分（type: "translation"）
- 展示翻译原文句子，支持句子切换
- 提供参考译文折叠面板
- 提供句子语法解析折叠面板
- 支持返回试卷选择列表

### FR-3: 作文指导模块完善
- 从真题JSON数据中提取作文部分（type: "writing"）
- 展示Part A应用文和Part B短文写作题目
- 提供参考范文折叠面板
- 提供写作技巧折叠面板
- 支持返回试卷选择列表

### FR-4: 长难句数据扩充与规范
- 从所有真题（2000-2025年，英语一+英语二）中提取长难句
- 数量目标：至少100句以上，越多越好（保证质量前提下）
- 每句包含：来源、原句、句子主干、中文翻译、详细语法分析
- 解析格式规范：参考标准考研英语长难句解析方式
- 按难度或来源分类展示

### FR-5: 做题模式工具栏
- 新增选择模式：默认模式，可以点击选择答案
- 新增书写模式：模拟手写答题环境
- 新增橡皮擦：清除已选答案
- 工具栏位置：题目区域顶部或底部

### FR-6: 真题精读题目交互修复
- 修复Text 2及以后页面切换报错问题
- 实现题目选项点击选择功能
- 显示选中状态（高亮/变色）
- 支持切换答案

## Non-Functional Requirements

- **NFR-1**: 界面风格统一，与现有设计语言保持一致
- **NFR-2**: 响应式布局，适配不同屏幕宽度
- **NFR-3**: 长难句解析准确规范，符合考研英语教学标准
- **NFR-4**: 页面加载流畅，无明显卡顿

## Constraints
- **Technical**: 纯前端实现，使用原生JavaScript，不引入新框架
- **Business**: 数据来源于已有的真题JSON文件
- **Dependencies**: 依赖现有pastPapers模块架构

## Assumptions
- 所有真题JSON文件中都包含翻译和作文部分数据
- 长难句可以从阅读文章中自动筛选（基于句子长度和复杂度）
- 做题模式工具栏可以通过CSS类切换实现

## Acceptance Criteria

### AC-1: 历年真题网格布局
- **Given**: 用户进入真题练习首页或任何功能的试卷选择页
- **When**: 查看历年真题列表
- **Then**: 试卷卡片以网格形式排列，一行多个，自动换行
- **Verification**: `human-judgment`

### AC-2: 翻译模块可正常使用
- **Given**: 用户在翻译练习中选择一套真题
- **When**: 页面加载完成
- **Then**: 显示翻译原文句子，可以切换句子，可以展开查看参考译文和解析
- **Verification**: `human-judgment`

### AC-3: 作文模块可正常使用
- **Given**: 用户在作文指导中选择一套真题
- **When**: 页面加载完成
- **Then**: 显示Part A和Part B作文题目，可以展开查看范文和写作技巧
- **Verification**: `human-judgment`

### AC-4: 长难句数量充足且解析规范
- **Given**: 用户进入长难句分析模块
- **When**: 浏览长难句列表
- **Then**: 长难句数量至少100句以上，每句都有规范的解析（主干、翻译、语法分析）
- **Verification**: `programmatic` + `human-judgment`

### AC-5: 做题模式工具栏可用
- **Given**: 用户在做题页面（完形填空或真题精读）
- **When**: 点击选择模式/书写模式/橡皮擦按钮
- **Then**: 对应模式生效，交互方式切换
- **Verification**: `human-judgment`

### AC-6: 真题精读题目可交互
- **Given**: 用户在真题精读页面查看题目
- **When**: 点击选项A/B/C/D
- **Then**: 选项被选中并高亮显示，可切换选择
- **Verification**: `human-judgment`

### AC-7: Text 2及以后页面可正常切换
- **Given**: 用户在真题精读页面
- **When**: 点击Text 2/Text 3/Text 4标签
- **Then**: 对应文章内容正确显示，无报错
- **Verification**: `programmatic`
